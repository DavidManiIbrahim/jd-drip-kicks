# JD Drip Kicks

Welcome to JD Drip Kicks, your premier destination for premium footwear. We offer a wide selection of shoes, sandals, slides, and crocs for both men and women.

## Design

This project uses a modern, responsive design built with:
- **Vite** for fast development and building.
- **React** as the UI library.
- **Tailwind CSS** for styling.
- **Shadcn UI** for high-quality, accessible components.

## Getting Started

To run this project locally:

1. Clone the repository
```sh
git clone <YOUR_GIT_URL>
```

2. Install dependencies
```sh
npm install
```

3. Start the development server
```sh
npm run dev
```

## Features

- **Premium Collection**: Browse our curated selection of top-quality footwear.
- **Responsive Design**: Shop comfortably on any device.
- **Modern UI**: Enjoy a smooth and visually appealing shopping experience.
